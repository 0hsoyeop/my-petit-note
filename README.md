### 목표

---
* Javascript로 HTML 요소 생성, 속성 변경하기
* Javascript로 로컬 스토리지 사용하기
* Bootstrap 사용하기

### 구현 기능

---
1. 메인 페이지
* 브라우저 로컬 스토리지에 저장된 글 목록을 보여준다.
* 글쓰기 버튼을 보여준다.

2. 글 상세 페이지
* 글 제목, 내용을 조회한다.
* 글 수정, 삭제 버튼을 보여준다.

3. 글 작성 페이지
* 제목, 내용 입력란을 만든다.
* 글 저장, 글쓰기 취소 버튼을 보여준다.
